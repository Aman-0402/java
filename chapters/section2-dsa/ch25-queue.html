Chapter 25: Queue

A queue is a First-In, First-Out (FIFO) data structure where items are added at the back and removed from the front вҖ” just like a ticket counter line.

рҹ“ң History: Queue theory was mathematically formalized by Danish engineer A.K. Erlang in 1909 while working for the Copenhagen Telephone Exchange. He needed to calculate how many telephone operators were needed to handle calls without long wait times. "Erlang" is now a unit of phone traffic вҖ” and also a programming language named after him.

The Ticket Counter Analogy

Imagine a queue at a railway ticket counter. The first person to arrive is the first person to be served. New arrivals join at the back. Nobody cuts in line (ideally). This is exactly how a queue data structure behaves: first in, first out.

The three core operations are:

  • enqueue (offer) вҖ” add an item to the back of the queue
  • dequeue (poll) вҖ” remove and return the item at the front
  • peek вҖ” look at the front item without removing it

Implementing a Queue in Java

Java's LinkedList class implements the Queue interface, making it the standard way to create a queue. Use the Queue interface type for the variable declaration.

import java.util.LinkedList;
import java.util.Queue;

Queue<String> queue = new LinkedList<>();

// offer() вҖ” add to back (returns false if queue is capacity-limited)
queue.offer("Alice");
queue.offer("Bob");
queue.offer("Carol");

// peek() вҖ” look at front without removing
System.out.println(queue.peek()); // Alice

// poll() вҖ” remove from front (returns null if empty)
System.out.println(queue.poll()); // Alice
System.out.println(queue.poll()); // Bob

System.out.println("Remaining: " + queue); // [Carol]
System.out.println("Size: " + queue.size()); // 1
рҹ’Ў Prefer offer() over add() and poll() over remove(). The offer/poll/peek methods return special values (false or null) on failure, while add/remove throw exceptions вҖ” the special-value versions are usually easier to work with in loops.
рҹ”Қ Fun Fact: Queues are fundamental to operating systems. When you open 5 apps at once, the OS puts their requests in a queue. The CPU processes one request at a time (or a few in parallel on multi-core), and the queue ensures no request is lost and all are eventually processed.

Use Cases for Queues

  • Print queue вҖ” documents print in the order they were submitted.
  • Task scheduling вҖ” OS processes CPU tasks in queue order.
  • Breadth-First Search (BFS) вҖ” explores a graph level by level using a queue.
  • Customer service simulation вҖ” customers wait in line to be served.
  • Message queues вҖ” systems like Kafka use FIFO queues for event streaming.

import java.util.LinkedList;
import java.util.Queue;

public class TicketQueue {

    public static void main(String[] args) {
        Queue<String> counter = new LinkedList<>();

        // Customers arrive and join the queue
        String[] arrivals = {"Alice", "Bob", "Carol", "Dave", "Eve"};
        for (String name : arrivals) {
            counter.offer(name);
            System.out.println(name + " joined the queue. Queue size: " + counter.size());
        }

        System.out.println("\n--- Counter opens ---");

        // Serve each customer in order
        while (!counter.isEmpty()) {
            String customer = counter.poll();
            System.out.println("Now serving: " + customer);

            // Peek at who is next (if anyone)
            if (!counter.isEmpty()) {
                System.out.println("  Next up: " + counter.peek());
            }
        }

        System.out.println("\nAll customers served. Counter closes.");
    }
}

Priority Queue вҖ” A Brief Introduction

A regular queue serves everyone in arrival order. A priority queue serves the highest-priority item first, regardless of order. Java's PriorityQueue class implements this using a heap internally.

import java.util.PriorityQueue;

// Default: smallest number = highest priority (min-heap)
PriorityQueue<Integer> pq = new PriorityQueue<>();
pq.offer(30);
pq.offer(10);
pq.offer(20);

System.out.println(pq.poll()); // 10 вҖ” smallest first
System.out.println(pq.poll()); // 20
System.out.println(pq.poll()); // 30
рҹҢҚ Real World: Uber's dispatch system uses a priority queue. When you request a ride, you're added to a queue. Nearby drivers receive requests from this queue. Priority is based on distance, surge pricing, and driver rating. Without queues, thousands of simultaneous requests would cause chaos.
в„№пёҸ Priority queues are widely used in Dijkstra's shortest path algorithm, hospital triage systems, and CPU task scheduling. They are covered in depth when you reach graph algorithms.
вҡ пёҸ PriorityQueue in Java does not guarantee the order of elements with equal priority. If you need stable ordering (ties broken by insertion order), you need a custom Comparator that includes a sequence number.
