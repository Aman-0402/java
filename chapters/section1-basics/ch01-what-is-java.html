📗 Section 1 — Java Basics

Chapter 1: What is Java?

Understand what Java is, why it was created, and why millions of developers use it today — before writing a single line of code.

The Simple Answer

Java is a programming language — a way for you to give instructions to a computer. You write instructions in Java, and the computer follows them.

Think of it like a recipe. You (the programmer) write the recipe (Java code). The computer (the chef) follows the recipe to produce a result (the running program).

Why Java?

There are hundreds of programming languages. Java is popular for three reasons:

  • Write Once, Run Anywhere — Java code runs on Windows, Mac, Linux, and Android without changes.
  • Jobs — Java is the #1 language for enterprise software and Android apps. High demand, high salaries.
  • Learning foundation — Java teaches you concepts (OOP, DSA) that transfer to any other language.
💡 Real-world Java: Gmail, Twitter's backend, LinkedIn, Netflix, and most Android apps are built with Java or Kotlin (Java's sibling language).

How Java Works (in 30 seconds)

Most languages translate your code directly to machine code for one specific operating system. Java does something different:

  1. You write .java source files.
  2. The Java Compiler (javac) translates them to bytecode (.class files).
  3. The Java Virtual Machine (JVM) runs that bytecode on any operating system.
ℹ️ Bytecode is not machine code. It is a middle format that the JVM understands. The JVM is different for each OS — but your bytecode stays the same. This is what "Write Once, Run Anywhere" means.

Java vs JavaScript — Are They the Same?

No. They sound similar but are completely different languages, created by different companies, for different purposes.

  • Java — general purpose, compiled, strongly typed, runs on JVM. Used for apps, backends, Android.
  • JavaScript — runs in browsers, loosely typed, interpreted. Used for web pages.
⚠️ Do not confuse them. Java and JavaScript share a name for historical marketing reasons only. They are as different as "car" and "carpet".
