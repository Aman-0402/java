📗 Section 1 — Java Basics

Chapter 6: Operators

Learn how to perform calculations, make comparisons, and combine conditions using Java's built-in operators.

What is an Operator?

An operator is a symbol that tells Java to perform a specific action on one or more values. Think of operators like the buttons on a calculator — + adds two numbers, - subtracts them. Java has operators for maths, for comparing values, for combining true/false conditions, and more.

Arithmetic Operators

These are the ones you know from school maths, plus one new one: the modulus operator %.

public class ArithmeticDemo {
    public static void main(String[] args) {
        int a = 17;
        int b = 5;

        System.out.println(a + b);  // 22  — addition
        System.out.println(a - b);  // 12  — subtraction
        System.out.println(a * b);  // 85  — multiplication
        System.out.println(a / b);  // 3   — integer division (see warning below!)
        System.out.println(a % b);  // 2   — remainder (17 = 3*5 + 2)
    }
}

The Integer Division Gotcha

When you divide two int values, Java throws away the decimal part and returns only the whole number. This surprises almost every beginner.

public class DivisionGotcha {
    public static void main(String[] args) {
        int x = 7;
        int y = 2;
        System.out.println(x / y);      // prints 3, NOT 3.5

        // Fix: use double (or cast one value to double)
        double result = (double) x / y;
        System.out.println(result);     // prints 3.5
    }
}
⚠️ 7 / 2 in Java gives 3, not 3.5. If you need decimal results, make at least one of the numbers a double, either by declaring it as double or by writing (double) x to cast it.

The Modulus Operator %

The % operator gives you the remainder after division. It sounds obscure, but it has many practical uses: checking if a number is even or odd, looping back around a range, checking divisibility.

🔍 Fun Fact: The modulus operator % is used to check if a number is even or odd (n % 2 == 0 = even). This single operation powers FizzBuzz — the most famous coding interview question ever used by Google, Facebook, and Amazon to screen candidates. Master % and you have already solved one of tech's most iconic interview problems.
public class ModulusDemo {
    public static void main(String[] args) {
        System.out.println(10 % 3);  // 1 (10 = 3*3 + 1)
        System.out.println(12 % 4);  // 0 (12 divides evenly by 4)
        System.out.println(9 % 2);   // 1 (9 is odd, any odd number % 2 = 1)
        System.out.println(8 % 2);   // 0 (8 is even, any even number % 2 = 0)
    }
}

Assignment Operators

You already know = (assign a value). The compound assignment operators are shortcuts that combine a maths operation with assignment.

public class AssignmentDemo {
    public static void main(String[] args) {
        int score = 10;

        score += 5;   // same as: score = score + 5;  -> 15
        score -= 3;   // same as: score = score - 3;  -> 12
        score *= 2;   // same as: score = score * 2;  -> 24
        score /= 4;   // same as: score = score / 4;  -> 6
        score %= 4;   // same as: score = score % 4;  -> 2

        System.out.println(score); // 2

        // Increment and decrement shortcuts
        score++;  // same as score = score + 1
        score--;  // same as score = score - 1
        System.out.println(score); // 2
    }
}
💡 i++ (increment) and i-- (decrement) are used so often in loops that they became their own operators. You will see them constantly in for loops in the next chapter.

Comparison Operators

Comparison operators compare two values and return a boolean — either true or false. They are the building blocks of decisions in your code.

public class ComparisonDemo {
    public static void main(String[] args) {
        int a = 10;
        int b = 20;

        System.out.println(a == b);  // false — equal to
        System.out.println(a != b);  // true  — not equal to
        System.out.println(a > b);   // false — greater than
        System.out.println(a < b);   // true  — less than
        System.out.println(a >= 10); // true  — greater than or equal to
        System.out.println(a <= 9);  // false — less than or equal to
    }
}
⚠️ = and == are completely different! = assigns a value (x = 5 puts 5 into x). == checks if two values are equal (x == 5 asks "is x equal to 5?"). Confusing them is one of the most common beginner bugs.

Logical Operators

Logical operators let you combine multiple conditions. Think of them like "AND", "OR", and "NOT" in plain English.

public class LogicalDemo {
    public static void main(String[] args) {
        int age = 20;
        boolean hasTicket = true;

        // && (AND) — both conditions must be true
        System.out.println(age >= 18 && hasTicket);   // true (is adult AND has ticket)

        // || (OR) — at least one condition must be true
        System.out.println(age >= 18 || hasTicket);   // true

        // ! (NOT) — flips true to false and vice versa
        System.out.println(!hasTicket);               // false

        // Practical example: can this person enter a concert?
        boolean canEnter = age >= 18 && hasTicket;
        System.out.println("Can enter: " + canEnter); // Can enter: true
    }
}

The Ternary Operator

The ternary operator is a compact way to write a simple if-else decision in a single line. The name "ternary" means it takes three parts: a condition, a value if true, and a value if false.

public class TernaryDemo {
    public static void main(String[] args) {
        int temperature = 28;

        // Syntax: condition ? valueIfTrue : valueIfFalse
        String weather = temperature > 25 ? "Hot" : "Cool";
        System.out.println("It is: " + weather); // It is: Hot

        int score = 72;
        String result = score >= 50 ? "Pass" : "Fail";
        System.out.println("Result: " + result); // Result: Pass
    }
}
ℹ️ The ternary operator is great for short, simple decisions. But if your condition or values are complex, use a full if-else statement instead — readability matters more than brevity.
🌍 Real World: Integer division (/) combined with modulus (%) is used in pagination on every website. If you have 101 items and show 10 per page: 101 / 10 = 10 pages, 101 % 10 = 1 item on the last page. Every "Page 1 of 11" you have ever seen was calculated with exactly these two operators.
📜 History: The && (short-circuit AND) and || (short-circuit OR) operators exist for performance. In 1970s computers, evaluating both sides of a boolean expression was expensive. Short-circuit evaluation — stopping as soon as the result is known — became standard across C, Java, JavaScript, and most modern languages.

🧠 Multiple Choice Questions

1. What does the % operator return for 17 % 5?

  • A) 3
  • B) 2
  • C) 3.4
  • D) 0
Show Answer

✅ B) 2 — 17 divided by 5 is 3 remainder 2. The % operator returns the remainder.

2. What is the result of 10 / 3 in Java (integer division)?

  • A) 3.33
  • B) 3
  • C) 4
  • D) 3.0
Show Answer

✅ B) 3 — When both operands are int, Java performs integer division (truncates the decimal). Use 10.0 / 3 for a decimal result.

3. Which operator checks if two values are equal?

  • A) =
  • B) =>
  • C) ==
  • D) !=
Show Answer

✅ C) == — Single = is assignment, == is comparison, != is "not equal".

4. What does x++ do?

  • A) Multiplies x by 2
  • B) Increments x by 1 after using its current value
  • C) Sets x to 1
  • D) Decrements x
Show Answer

✅ B) Increments x by 1 after using its current value — x++ is post-increment. ++x increments before use.
