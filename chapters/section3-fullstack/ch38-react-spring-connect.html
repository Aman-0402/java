Chapter 38: Connect React + Spring Boot

Make your React frontend talk to your Spring Boot backend — the bridge that completes your full-stack application.

The Setup

In full-stack development, two servers run at the same time:

  • Spring Boot — runs on http://localhost:8080, serves your REST API
  • React — runs on http://localhost:3000, serves your UI

React fetches data from Spring Boot using JavaScript's built-in fetch() function (or Axios, a popular library).

Step 1: Enable CORS in Spring Boot

By default, browsers block requests from localhost:3000 to localhost:8080 because they are different origins. You must tell Spring Boot to allow it:

@RestController
@RequestMapping("/api/todos")
@CrossOrigin(origins = "http://localhost:3000")  // allow React dev server
public class TodoController {
    // ...
}
💡 For production, replace "http://localhost:3000" with your actual frontend domain, e.g. "https://myapp.netlify.app".

Step 2: Fetch Data in React

Create a React component that loads todos from your Spring Boot API when it mounts:

import { useState, useEffect } from 'react';

function TodoList() {
  const [todos, setTodos] = useState([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    fetch('http://localhost:8080/api/todos')
      .then(res => {
        if (!res.ok) throw new Error('Server error: ' + res.status);
        return res.json();
      })
      .then(data => {
        setTodos(data);
        setLoading(false);
      })
      .catch(err => {
        setError(err.message);
        setLoading(false);
      });
  }, []); // empty array = run once on mount

  if (loading) return <p>Loading...</p>;
  if (error)   return <p>Error: {error}</p>;

  return (
    <ul>
      {todos.map(todo => (
        <li key={todo.id}>
          {todo.completed ? '✅' : '⬜'} {todo.title}
        </li>
      ))}
    </ul>
  );
}

export default TodoList;

Step 3: Create a Todo from React

function AddTodo({ onAdd }) {
  const [title, setTitle] = useState('');

  const handleSubmit = async (e) => {
    e.preventDefault();
    const res = await fetch('http://localhost:8080/api/todos', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ title, completed: false })
    });
    const newTodo = await res.json();
    onAdd(newTodo);
    setTitle('');
  };

  return (
    <form onSubmit={handleSubmit}>
      <input
        value={title}
        onChange={e => setTitle(e.target.value)}
        placeholder="New todo..."
      />
      <button type="submit">Add</button>
    </form>
  );
}
⚠️ Always check res.ok before calling res.json(). If the server returns a 404 or 500 error, calling res.json() on an error response can throw confusing errors.

Putting It Together in App.js

import { useState, useEffect } from 'react';

function App() {
  const [todos, setTodos] = useState([]);

  useEffect(() => {
    fetch('http://localhost:8080/api/todos')
      .then(r => r.json())
      .then(setTodos);
  }, []);

  const addTodo = async (title) => {
    const res = await fetch('http://localhost:8080/api/todos', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ title, completed: false })
    });
    const todo = await res.json();
    setTodos(prev => [...prev, todo]);
  };

  return (
    <div>
      <h1>My Todos</h1>
      <input
        onKeyDown={e => e.key === 'Enter' && addTodo(e.target.value)}
        placeholder="Press Enter to add..."
      />
      <ul>
        {todos.map(t => <li key={t.id}>{t.title}</li>)}
      </ul>
    </div>
  );
}

export default App;

✅ Try It

Run Spring Boot on port 8080 and React on port 3000 simultaneously. Open React in the browser, add a todo via the input, and verify it appears in the list. Then check http://localhost:8080/api/todos to confirm the data was saved to the backend.
